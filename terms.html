Description of the Program

Personal Discord SMS Notifications is an automated application-to-person (A2P) notification system operated by a single individual for their own personal use. The application monitors a private Discord server owned by the operator and forwards new messages, from designated channels, as SMS to the operator's own verified mobile number. There are no external users or customers.

Recipients

Messages are sent exclusively to the operator's own phone number. No third party receives messages.

Message Frequency

Variable, dependent on Discord activity in the monitored channels. Typically fewer than 50 messages per day.

Message and Data Rates

Standard carrier message and data rates may apply.

Keywords

  • HELP — Reply HELP for assistance. The operator will receive a standard Twilio help response.
  • STOP — Reply STOP to opt out and stop all further messages. Opt-out is handled by Twilio's standard STOP filter.

Opt-Out

The operator can stop messages at any time by:

  • Replying STOP to any message
  • Removing the configured phone number from the application's environment variables
  • Stopping or deleting the application deployment
